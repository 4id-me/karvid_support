# Sample karaoke videos for App Review

Drop a short karaoke clip here named `karvid_sample_karaoke.mp4`.

Recommended:
- Length: 30s–2 min (reviewer needs only a short test)
- Format: H.264 MP4, vertical or horizontal both fine
- Size: under 25 MB (GitHub Pages serves comfortably)
- Content: a track you own the rights to (your own Suno output, public-domain song, or CC0/CC-BY karaoke). Apple's reviewers will scrutinize copyright on hosted media.

After dropping the file:

    git add sample_videos/karvid_sample_karaoke.mp4
    git commit -m "Add sample karaoke video for App Review"
    git push
